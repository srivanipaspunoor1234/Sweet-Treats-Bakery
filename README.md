Sweet Treats Bakery Website

A static bakery-themed website designed using HTML and CSS, showcasing delicious bakery items with a clean layout, product cards, and interactive hover effects.

~Project Overview
Sweet Treats Bakery is a frontend web project that highlights bakery products such as cupcakes, cakes, croissants, and cookies. The website focuses on visual appeal, structured content, and simple user interaction.

~ Features
- Attractive header and navigation menu  
- Bakery product cards with images and descriptions  
- Hover effects on cards and buttons  
- Promotional scrolling offer section  
- Clean and user-friendly layout  

~ Technologies Used
- HTML5  
- CSS3
